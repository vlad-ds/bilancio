Key Metrics

  • δ (Delta): Default rate - fraction of debt not settled on time (lower is better)
  • φ (Phi): Clearing rate - fraction of debt settled on due day (higher is better)
  • Trading Effect: δ_passive - δ_active (positive = dealer reduced defaults)
  • κ (Kappa): Liquidity ratio L₀/S₁ (higher = more liquid system)
  • c (Concentration): Debt distribution inequality (lower = more unequal)
  • μ (Mu): Maturity timing skew (0 = early, 1 = late due dates)
